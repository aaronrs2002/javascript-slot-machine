#slot machine - javascript css "sprite" tutorial

In this tutorial we will walk through the CSS and HTML steps to creating a sprite. Then, we walk through a working example where we use sprite in a basic JavaScript game.

What is a CSS sprite: an image with many small icons on a page
Why are they useful: you only load one image and re-utilize it over and over
